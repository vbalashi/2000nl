# Splash Source

These `apple-touch-startup-image` PNGs are generated locally using ImageMagick.

Design:
- Background: `#F5F7FB`
- Primary: `#2e2bee`
- Text: `#0f172a`
- Font: Noto Sans (Black)

Files in this folder are referenced from `apps/ui/app/layout.tsx` via `rel="apple-touch-startup-image"` with device-specific media queries.
